chai-calling-with
=================

> Readable way for testing exceptions for [Chai](http://chaijs.com)

## Installation

Install using [npm](https://www.npmjs.org/):

```sh
npm install chai-calling-with
```

## Setup
```javascript
var chai = require('chai');
chai.use(require('chai-calling-with'));
```
